# To-Do List Website

## Description
A simple collaborative To-Do List built using HTML, CSS, and JavaScript.

## Features
- Add tasks
- Delete tasks
- Mark tasks as completed

## Collaboration
- Used GitHub for version control
- Feature branches
- Pull requests
- Merge conflict handling

## Team Members
- Person 1: UI (HTML, CSS)
- Person 2: Logic (JavaScript)

## How to Run
Open index.html in browser
